# Hero Section Prompt

Create a hero section for a gardening/landscaping website with the following
specifications:

## Layout

- Full-width section with a dark background (dark gray/black, approximately
  #1a1a1a or similar)
- Two-column layout: content on the left (60%), image grid on the right (40%)
- Responsive design that stacks on mobile devices

## Content Area (Left Side)

### Heading

- Large, bold heading with the text: "Designing landscapes that grow with
  beauty, and last for generations."
- The phrase "Designing landscapes" should be in bright green color (similar to
  #4ade80 or #22c55e)
- The rest of the heading should be in white
- Typography: Modern sans-serif font, very large size (approximately 3-4rem)
- Line height should be generous for readability

### Subheading

- Gray text with the message: "Transforming lawns, gardens, and landscapes into
  vibrant, sustainable spaces that enhance beauty and bring nature closer to
  you."
- Slightly smaller font size (approximately 1.125rem)
- Light gray color for contrast against dark background

### Call-to-Action Buttons

- Two buttons side by side
- Primary button: "Book An Appointment" with a calendar/sparkle icon
  - Bright green background matching the heading color
  - White text
  - Rounded corners
  - Hover effects
- Secondary button: "Contact Us" with an arrow icon
  - Transparent/outlined style
  - White text and border
  - Hover effects

## Image Grid (Right Side)

- Grid of 4 images arranged in 2x2 layout
- Images should show:
  1. Top-left: Wooden pergola with landscaped garden area
  2. Top-right: Person in plaid shirt holding gardening equipment
  3. Bottom-left: Close-up of hands trimming a hedge with garden shears
  4. Bottom-right: Person using a hedge trimmer on bushes
- Images should have rounded corners
- Slight gap/spacing between images
- Images should be high quality and professional

## Styling Details

- Section should have generous padding (top and bottom)
- Content should be centered vertically
- Smooth transitions and animations on hover states
- Ensure proper contrast ratios for accessibility
- Use CSS Grid or Flexbox for responsive layout

## Technical Requirements

- Built with React/Next.js components
- Use Tailwind CSS for styling
- Implement responsive breakpoints
- Add smooth scroll behavior if linked from navigation
- Optimize images for web performance
- Include proper alt text for images
- Use semantic HTML elements (section, h1, p, etc.)
